# 变更：创建工业自动化智能体平台

## 原因
基于 LibreChat 项目基础，在 `industryclient` 目录下构建一个工业自动化智能体平台。该平台将提供完整的工业场景解决方案，包括文档管理、设备连接、数据采集、第三方系统集成、业务流程设计等功能，满足工业 4.0 智能制造需求。

## 变更内容
- **新增 industryclient 前端项目**：基于 React/Vite/TypeScript 技术栈
- **复用 client 现有组件**：UI 组件库、认证系统、文件处理功能
- **第一阶段实施核心功能**：文档管理、设备管理、数据管理
- **第二阶段实施扩展功能**：第三方集成、业务流程设计、界面设计器

## 影响
- **新增目录结构**：
  - `industryclient/`: 工业自动化前端项目
  - `api/models/`: 新增工业数据模型（设备、数据点位、配置等）
  - `api/routes/`: 新增工业相关 API 路由
  - `api/services/`: 新增工业数据处理服务

- **受影响的规范**：
  - 新增：工业自动化功能规范

- **受影响的代码**：
  - `industryclient/`: 全新项目目录
  - `api/models/`: 新增设备、数据点位、系统配置等数据模型
  - `api/routes/industry.js`: 新增工业功能 API 路由
  - `api/services/Industry/`: 新增工业数据处理服务
